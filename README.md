# NothingOs (Simple C# Kernel)

A basic command-line operating system built with **C#** and the **Cosmos DevKit**. This was developed as an educational project to understand core kernel loops and simple file system management.

## 🛠 Features
* **File Management:** Create files, rename files, and navigate directories (`cd`).
* **Basic Math:** Integrated calculator for simple arithmetic (addition, subtraction, multiplication, division).
* **CLI Interface:** A standard command-line interface for user interaction.

## 📂 Project Structure
* `Kernel.cs`: The main entry point and system loop.
* `Task.cs`: Handles specific command logic.
* `Project_ReportOS.pdf`: Documentation and project details.
